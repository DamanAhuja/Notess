# 1. 🧠 What is Machine Learning?

### 💡 Definition

Machine Learning (ML) is a branch of Artificial Intelligence where systems learn patterns from data and improve performance without being explicitly programmed.

---

# 🔥 Simple Meaning

Instead of writing exact rules:
👉 we give data
👉 system learns patterns automatically.

---

# 🔥 Example

Spam Detection:

* Give emails labeled spam/not spam
* Model learns patterns
* Predicts future spam emails

---

# 🔥 Viva

👉 “Difference between traditional programming and ML?”

Traditional programming:

* rules + data → output

Machine learning:

* data + output → learns rules

# 3. 📏 FEATURE SCALING (MOST IMPORTANT)

EXTREMELY IMPORTANT.

---

# 💡 Definition

Feature scaling means transforming features so they lie within similar ranges.

---

# 🔥 Why Needed?

Some algorithms depend on distance:

* KNN
* K-Means
* SVM

Large-valued features dominate smaller ones.

---

# 🔥 Example

| Feature | Range           |
| ------- | --------------- |
| Age     | 20–50           |
| Salary  | 20,000–1,00,000 |

Salary dominates.

---

# 🔥 Viva

👉 “Why feature scaling important?”

To prevent large-range features from dominating learning.

---

# 4. 🔹 TYPES OF FEATURE SCALING

VERY IMPORTANT.

---

# 🔹 Normalization (Min-Max Scaling)

Scales data between:

```text id="6ghbxr"
0 and 1
```

Formula:
x' = \frac{x - x_{min}}{x_{max} - x_{min}}

---

# 🔥 Advantages

* Simple
* Bounded range

---

# 🔹 Standardization (Z-Score Normalization)

Transforms data with:

* mean = 0
* standard deviation = 1

Formula:

genui{"math_block_widget_always_prefetch_v2":{"content":"z = \frac{x - \mu}{\sigma}"}}

---

# 🔥 Advantages

Works well for normally distributed data.

---

# 🔥 VERY IMPORTANT DIFFERENCE

| Normalization         | Standardization  |
| --------------------- | ---------------- |
| Scales between 0–1    | Mean = 0, SD = 1 |
| Sensitive to outliers | More robust      |

---

# 🔥 Viva

👉 “Difference between normalization and standardization?”

Normalization scales data to fixed range, while standardization transforms data around mean 0 and standard deviation 1.

# 5. 🎯 COST FUNCTION

VERY IMPORTANT.

---

# 💡 Definition

Measures prediction error of model.

---

# 🔥 Goal

Minimize cost/error.

---

# 🔥 Common Cost Function

Mean Squared Error (MSE)

MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2

---

# 🔥 Meaning

Difference between:

* actual value
* predicted value

---

# 🔥 Viva

👉 “Why square errors in MSE?”

To avoid negative cancellation and penalize large errors.

---

# 6. 📉 GRADIENT DESCENT (MOST IMPORTANT)

EXTREMELY IMPORTANT.

---

# 💡 Definition

Gradient Descent is an optimization algorithm used to minimize the cost function.

---

# 🔥 Core Idea

Move step-by-step toward minimum error.

---

# 🔥 Update Rule

\theta := \theta - \alpha \frac{\partial J(\theta)}{\partial \theta}

Where:

* θ → parameter
* α → learning rate
* J(θ) → cost function

---

# 🔥 Simple Meaning

Adjust model parameters repeatedly to reduce error.

---

# 🔥 Learning Rate

Controls step size.

---

# 🔥 If Learning Rate Too Large

May overshoot minimum.

---

# 🔥 If Too Small

Training becomes slow.

---

# 🔥 Viva

👉 “Purpose of gradient descent?”

To minimize model error/cost.

---

# 10. 📊 REGRESSION EVALUATION METRICS

# 🔹 Mean Squared Error (MSE)

Average squared error.

---

# 🔹 Root Mean Squared Error (RMSE)

Square root of MSE.

RMSE = \sqrt{MSE}

---

# 🔹 R² Score

Measures goodness of fit.

Range:

```text id="srr8t4"
0 to 1
```

Higher value → better model.

---

# 🔥 Viva

👉 “Which metric penalizes large errors more?”

# 11. 📊 CLASSIFICATION EVALUATION METRICS

EXTREMELY IMPORTANT.

---

# 🔹 Accuracy

### 💡 Definition

Percentage of correct predictions.

Formula:
Accuracy = \frac{TP+TN}{TP+TN+FP+FN}

---

# 🔹 Precision

### 💡 Meaning

How many predicted positives are actually correct.

Precision = \frac{TP}{TP+FP}

---

# 🔹 Recall

### 💡 Meaning

How many actual positives identified correctly.

Recall = \frac{TP}{TP+FN}

---

# 🔹 F1-Score

Balance of precision and recall.

F1 = 2\cdot\frac{Precision \cdot Recall}{Precision+Recall}

---

# 🔥 VERY IMPORTANT TERMS

| ---- | -------------- |

| TN   | True Negative  |
| FP   | False Positive |
| FN   | False Negative |

# 1. 🧠 What is Clustering?

### 💡 Definition

Clustering is an unsupervised learning technique used to group similar data points together.

---

# 🔥 Simple Meaning

Data points with similar properties are placed in the same group (cluster).

---

# 🔥 Example

Customer segmentation:

* students
* professionals
* senior citizens

---

# 🔥 Goal

Find hidden patterns/structures in unlabeled data.

---

# 🔥 Viva

👉 “Difference between classification and clustering?”

Classification:

* labeled data

Clustering:

* unlabeled data

---

# 2. 🔷 CHARACTERISTICS OF CLUSTERING

---

# 🔹 Intra-Cluster Similarity

Points within same cluster should be similar.

---

# 🔹 Inter-Cluster Dissimilarity

Different clusters should be distinct.

---

# 🔥 Example

Students with similar marks grouped together.

---

# 3. 📏 DISTANCE MEASURES (VERY IMPORTANT)

EXTREMELY IMPORTANT.

Distance metrics determine similarity between data points.

---

# 🔹 Euclidean Distance

MOST COMMON.

Measures straight-line distance.

Formula:
d = \sqrt{\sum_{i=1}^{n}(x_i-y_i)^2}

---

# 🔥 Example

Distance between two points in graph.

---

# 🔹 Manhattan Distance

Measures grid-like distance.

Formula:
d = \sum_{i=1}^{n}|x_i-y_i|

---

# 🔥 Example

Distance travelled in city blocks.

---

# 🔹 Minkowski Distance

Generalized distance metric.

Euclidean distance.

# 4. 🎯 K-MEANS CLUSTERING (MOST IMPORTANT)

EXTREMELY IMPORTANT.

---

# 💡 Definition

K-Means is an unsupervised clustering algorithm that partitions data into K clusters.

---

# 🔥 Core Idea

Each cluster has a centroid.

Data points assigned to nearest centroid.

---

# 🔥 Steps of K-Means

VERY IMPORTANT.

---

## Step 1

Choose number of clusters:

```text id="cjlwm6"
K
```

---

## Step 2

Initialize random centroids.

---

## Step 3

Assign data points to nearest centroid.

---

## Step 4

Recalculate centroids.

---

## Step 5

Repeat until centroids stabilize.

---

# 🔥 Example

Grouping customers by:

* income
* spending habits

---

# 🔥 Advantages

* Simple
* Fast
* Easy implementation

---

# 🔥 Disadvantages

* Need to choose K manually
* Sensitive to initialization
* Affected by outliers

---

# 🔥 Viva

👉 “What is centroid?”

Center point of a cluster.

---

# 🔥 Viva

👉 “Why K-Means sensitive to outliers?”

Outliers shift centroids.
